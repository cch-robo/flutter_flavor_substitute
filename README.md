# flavor_substitute

Flutter 疑似 flavor ライブラリ.

## Getting Started

InttelliJ IDEA 系の Before lunch オプションを使い、
疑似 flavor 指定による debug/staging/release環境 ごとにリソースを切り替える
コンセプト・ライブラリです。
